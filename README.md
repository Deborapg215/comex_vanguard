📌 Sobre o Projeto

O **COMEX Vanguard** foi criado para otimizar o fluxo operacional de empresas de Comércio Exterior, reduzindo processos manuais e integrando informações financeiras, operacionais e fiscais em uma única plataforma.

O sistema permite controlar desde o cadastro dos processos até a emissão das Invoices e Notas Fiscais, proporcionando maior produtividade e rastreabilidade.

---

## ✨ Funcionalidades

- 📦 Gestão de Processos (HBL)
- 👥 Cadastro de Clientes
- 💰 Emissão de Invoices
- 🧾 Emissão de Notas Fiscais
- 📄 Controle de Serviços
- 📊 Dashboard Gerencial
- 🔎 Pesquisa de Processos
- 📈 Indicadores Operacionais
- 🔐 Autenticação de Usuários
- 👤 Controle de Perfis e Permissões
- 📂 Upload de Arquivos
- ☁️ Banco de Dados em Nuvem
- 📱 Interface Responsiva
